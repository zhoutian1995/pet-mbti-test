# doc-locator - 文档定位器

> 📂 帮你快速找到文档应该存放的位置，避免重复和乱放

**版本**: 1.0
**创建时间**: 2026-03-07

---

## 🎯 用途

1. **定位文档位置** - 告诉你对应类型的文档应该放在哪里
2. **检查重复文档** - 写新文档前检查是否已存在
3. **提供模板参考** - 给出文档结构建议

---

## 📋 文档命名规范

### 日期格式
- 日报：`YYYY-MM-DD-日报.md` → `2026-03-07-日报.md`
- 周报：`YYYY-Www-周报.md` → `2026-W10-周报.md`
- 月报：`YYYY-MM-月报.md` → `2026-03-月报.md`
- 报告：`YYYY-MM-DD-{主题}报告.md` → `2026-03-07-架构检查报告.md`

### 通用规则
- 使用中文命名
- 日期在前便于排序
- 主题简明扼要
- 避免特殊字符

---

## ⚠️ 常见错误

| 错误 | 正确做法 |
|------|---------|
| API Key 分散在多个文档 | 统一放到 `09-私人/API 密钥管理总表` |
| 日报不写日期 | 格式：`YYYY-MM-DD-日报.md` |
| 报告没有主题 | 格式：`YYYY-MM-DD-{主题}报告.md` |
| 项目文档乱放 | 统一放到 `02-项目/{项目名}/` 下 |
