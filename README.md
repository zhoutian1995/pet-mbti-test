# mbti-test

`mbti-test` 是一个宠物 MBTI 测试项目，支持猫和狗两条测试路径，用问答方式生成宠物性格类型、常见品种、相处建议和可分享结果页。

它不是严肃心理测评，而是一个偏传播、娱乐和社交分享的互动网页产品。

## 这个项目解决什么问题

- 给宠物主人一个有趣的互动测试入口。
- 把猫狗行为问题包装成易传播的 MBTI 结果。
- 生成适合分享的结果页，提高传播和复访。
- 可作为宠物内容、社群活动或小程序产品的原型。

## 项目结构

- `index.html`：项目入口。
- `cat/`：猫测试页面、题库和结果数据。
- `dog/`：狗测试页面、题库和结果数据。
- `result.html`：结果展示页。
- `assets/`：图片和静态资源。
- `TEST_CASES.md`：测试用例。
- `AGENTS.md` 等文件：项目上下文和协作说明。

## 当前能力

- 猫/狗分物种测试。
- 中英文题库。
- 支持“不确定/没遇到过”选项。
- 结果页展示 MBTI 类型、描述、标签、语录和相处指南。
- 具备分享按钮和重新测试入口。

## 注意事项

- 当前仓库里有未提交的题库和结果页改动，提交前应确认内容版本。
- `data.db` 是本地数据文件，不一定需要进入 Git。
- `node_modules/` 是依赖目录，不应作为主要代码阅读入口。
