# 中英文切换实现方案

## 方案设计

### 1. URL 参数切换
- 使用 `?lang=en` 或 `?lang=zh` 切换语言
- 默认中文

### 2. 文件结构

```
mbti-test/
├── index.html (首页)
├── cat/
│   ├── index.html (测试页)
│   ├── questions.json (题目数据)
│   └── questions.en.json (英文题目数据)
│   └── results.json (结果数据)
│   └── results.en.json (英文结果数据)
├── dog/
│   ├── index.html (测试页)
│   ├── questions.json (题目数据)
│   └── questions.en.json (英文题目数据)
│   └── results.json (结果数据)
│   └── results.en.json (英文结果数据)
├── result.html (结果页)
└── i18n.js (语言切换工具)
```

### 3. 功能清单

| 页面 | 需要添加 |
|------|----------|
| 首页 | 语言切换按钮 |
| 猫咪测试页 | 语言切换按钮 + 英文数据加载 |
| 狗狗测试页 | 语言切换按钮 + 英文数据加载 |
| 结果页 | 语言切换按钮 + 英文数据加载 |

### 4. 语言切换按钮样式

```
[中/EN] - 点击切换
```

位置：右上角
